# Modern Login & Sign Up UI

A modern and responsive single-page authentication interface built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Login and Sign Up on a single page
- Smooth form transitions
- Show/Hide password
- Remember me option
- Forgot password link
- Social login buttons
- Form validation

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Download or clone the repository.
2. Open `index.html` in your browser.
